About This Project

TaxBot AI is an intelligent tax filing assistant built to simplify how Indian taxpayers understand and file their income taxes. Rather than replacing professional advice, this tool serves as a companion that walks you through complex decisions using data-driven insights.

This project was developed as part of the Artificial Intelligence coursework (Semester 4), demonstrating practical applications of seven distinct AI techniques working together in a single, cohesive platform. From machine learning classification to natural language understanding, every component is built from scratch using real-world tax data and Indian tax regulations for FY 2024-25.

AI Models & Architecture

Each model serves a specific purpose in the tax filing pipeline. Here is how they work together.

A* State-Space Search

Search Algorithm

Models the tax filing process as a graph where each state represents a filing step and edges are transitions. Uses an admissible heuristic based on data completeness to find the optimal, lowest-cost filing path from start to submission.

TechniqueA* Search with Priority Queue
HeuristicData completeness scoring
Use CaseFiling path optimization

ML Tax Regime Classifier

Classification

An ensemble of Random Forest and Gradient Boosting classifiers that predicts whether the Old or New tax regime is optimal for a given taxpayer. Trained on synthetic taxpayer profiles with features including income level, deductions, HRA, and investment patterns.

ModelsRandom Forest + Gradient Boosting
FeaturesIncome, deductions, HRA, investments
Use CaseRegime recommendation

Tax Liability Predictor

Regression

A Gradient Boosting Regressor that predicts expected tax liability with confidence intervals. Uses engineered features like log-income transformations and deduction-to-income ratios to capture non-linear tax slab boundaries.

ModelGradient Boosting Regressor
EngineeringLog-income, deduction ratios
Use CaseTax amount prediction

Bayesian Network

Probabilistic Reasoning

Implements Bayes' theorem with Conditional Probability Tables to assess audit risk. Calculates the probability of an audit given income, deduction patterns, and chosen regime using likelihood modifiers derived from Indian tax norms.

TechniqueBayes' Theorem + CPTs
InputsIncome, deductions, regime
Use CaseAudit risk assessment

Document Classifier

Deep Learning

A Multi-Layer Perceptron neural network with a 256-128-64 architecture, ReLU activations, dropout regularization, and early stopping. Classifies uploaded tax documents into seven categories including Form 16, salary slips, and bank statements.

ArchitectureMLP (256 → 128 → 64)
ActivationReLU + Dropout
Use CaseDocument classification

NLP Chatbot Engine

Natural Language Processing

A full NLP pipeline that processes user queries through tokenization, TF-IDF embeddings, a custom attention mechanism, and neural intent classification. Extracts entities like income amounts and deduction sections, then generates contextual responses.

PipelineTF-IDF → Attention → MLP
Intents25+ tax-specific categories
Use CaseConversational assistant

Generative AI Engine

Generative AI

Uses template-based generation with few-shot prompting, role-based personas (tax expert), and chain-of-thought reasoning to produce human-readable tax summaries, step-by-step explanations, and personalized filing recommendations.

TechniquesFew-shot, CoT, role-based
OutputTax summaries & explanations
Use CaseReport generation

Deduction Recommender

Heuristic Optimization

A greedy heuristic optimizer that scans your current deduction usage against all available sections under Indian tax law. It identifies untapped savings opportunities and ranks them by marginal tax rate impact for maximum benefit.

StrategyGreedy optimization
Sections80C, 80D, 80CCD, 80G, 24(b)
Use CaseSavings maximization
